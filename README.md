# 高齢者救急 退院支援クイズ ― 医療ソーシャルワーク学習アプリ

救命救急センターに勤務する医療従事者・研修医・医療ソーシャルワーカー(MSW)向けに、突然の傷病で救急搬送された高齢者の退院支援において必要となる公的支援・社会資源を学習するためのWebアプリです。

## 概要

超高齢化社会において、救急搬送される高齢者の多くは、急性期治療が終わった後、直ちに在宅復帰できないことがしばしばあります。本人・家族の状況(独居、疎遠な家族、認知症、経済的困窮、老老介護など)に応じて、適切な公的支援・社会資源を組み合わせる必要があります。本アプリは、典型的な5シナリオを通じて、MSWの介入ポイントと制度選択の考え方をクイズ形式で学べる教材です。

加えて、日本で利用可能な高齢者向け公的支援を分類した**リファレンス**も内蔵しており、学習・実務の両面で参照できます。

## 主な機能

### 1. 事例クイズ(5シナリオ)
すべて「救命救急センターに搬送された、それまでADL良好(または要支援程度)の高齢者」という設定で統一されています。

| # | シナリオ | 学習ポイント |
|---|---|---|
| 1 | 75歳男性・独居・遠方に疎遠な家族 / 交通外傷で救命搬送 → 大腿骨頸部骨折 → リハ転院 → 自宅退院 | インテーク、キーパーソン設定、自賠責、回復期リハ、暫定ケアプラン、住宅改修、地域連携 |
| 2 | 82歳女性・独居・元々ADL自立 / 重症肺炎・敗血症性ショックで救命搬送 → 認知症が顕在化 | 権利擁護、成年後見、日常生活自立支援事業、認知症初期集中支援チーム、認知症対応型サービス |
| 3 | 78歳男性・要支援1 / 老老介護中にくも膜下出血で救命搬送 | 残された要介護者の緊急支援、ショートステイ緊急利用、レスパイト、身体障害者手帳、回復期リハ |
| 4 | 70歳男性・国民年金のみ・独居 / 急性心筋梗塞・心原性ショックで救命搬送 | 生活保護、無料低額診療、高額療養費、住居確保給付金、心臓リハ、訪問薬剤管理、心臓機能障害の手帳 |
| 5 | 78歳女性・元々ADL自立 / 消化管穿孔で救命搬送 → 進行胃癌判明 → 自宅看取り希望 | 末期がんの介護保険申請、訪問診療・訪問看護(医療保険)、ACP、退院前カンファレンス、グリーフケア |

各シナリオは複数の設問(単一選択/複数選択)で構成され、回答送信後に正誤の色分け表示と詳細な解説が表示されます。

### 2. 公的支援制度リファレンス
日本で利用可能な高齢者向けの公的支援・社会資源を **8カテゴリー** に分類:

1. **経済的支援**(医療費・生活費・住居費):高額療養費、生活保護、無料低額診療、自賠責、労災、障害年金、難病助成 など
2. **医療継続支援**(在宅医療・リハ):訪問診療、訪問看護、回復期リハ病棟、地域包括ケア病棟、緩和ケア病棟 など
3. **身体的・介護支援**(介護保険サービス):要介護認定申請、ケアマネジャー、訪問介護、デイ、ショート、小多機、看多機、福祉用具、住宅改修 など
4. **住まい・施設**:特養、老健、介護医療院、グループホーム、サ高住、有料老人ホーム、養護老人ホーム、公営住宅、住居確保給付金
5. **家族・介護者支援**:レスパイト、家族会、介護休業、ヤングケアラー支援、グリーフケア、退院前カンファ
6. **権利擁護・意思決定支援**:成年後見、任意後見、日常生活自立支援事業、ACP、意思決定支援ガイドライン、高齢者虐待防止法、各種障害者手帳
7. **地域連携・相談窓口**:地域包括支援センター、福祉事務所、保健所、認知症初期集中支援チーム、認知症疾患医療センター、民生委員、社協、難病・がん相談支援センター
8. **訪日外国人旅行者(短期滞在者)への支援**:海外旅行保険、本国大使館、医療通訳、JMIP、JNTOコールセンター、未払医療費対応、本国搬送、在留資格 など

各項目には以下の情報が掲載されています:
- **概要(desc)**:制度の説明
- **提供主体(provider)**:国・都道府県・市町村・民間など、誰が運営しているか
- **申請窓口(where)**:具体的にどこで申請・相談すればよいか

## 使い方

### ローカル実行
特別なセットアップは不要です。`index.html` を任意のWebブラウザで開くだけで動作します。

```bash
# macOSの場合
open index.html

# または、ファイルをダブルクリック
```

### 操作方法
1. **トップ画面**で、いずれかの事例カードをクリックするとクイズが開始されます。
2. 設問の選択肢をクリックし、複数選択の設問では該当するものをすべて選択してください。
3. **「回答する」**ボタンを押すと正誤判定と解説が表示されます。
4. **「次の設問へ」**で次に進み、最後まで回答すると結果画面で完全正解数が表示されます。
5. トップ画面の **「📚 公的支援制度リファレンスを開く」** ボタンから、いつでも制度一覧を参照できます。

## ファイル構成

```
MSW_quiz/
├── index.html      # アプリ本体(HTML/CSS/JavaScriptすべて含む単一ファイル)
└── README.md       # 本ファイル
```

シナリオデータ・リファレンスデータはすべて `index.html` 内の JavaScript オブジェクトとして定義されており、編集により容易に追加・修正が可能です。

## カスタマイズ

### 新しいシナリオを追加する
`index.html` 内の `const scenarios = [...]` 配列に、以下の形式で追加してください:

```javascript
{
  id: 6,
  title: "シナリオ6:○○",
  short: "短い説明",
  body: `症例の詳細...`,
  questions: [
    {
      q: "Q1. 設問文",
      type: "multi",  // または "single"
      options: [
        { text: "選択肢1", correct: true },
        { text: "選択肢2", correct: false }
      ],
      explain: `<strong>正解:1</strong><br>解説文...`
    }
  ]
}
```

### リファレンスに項目を追加する
`const referenceData = [...]` 配列の該当カテゴリに以下の形式で追加してください:

```javascript
{
  name: "制度名",
  desc: "制度の概要",
  provider: "提供主体(国/都道府県/市町村/民間など)",
  where: "申請窓口"
}
```

## 技術仕様

- **フロントエンド**:純粋な HTML / CSS / JavaScript(フレームワーク不使用)
- **依存関係**:なし(オフライン動作可)
- **ブラウザ要件**:モダンブラウザ(Chrome、Safari、Firefox、Edge の最新版)
- **言語**:日本語

## 注意事項

- 本アプリは **学習目的のシミュレーション教材** です。
- 制度の対象要件・手続き・名称・上限額等は、自治体や年度によって異なる場合があります。実際の運用にあたっては、必ず**最新の法令・厚生労働省通知・自治体情報**をご確認ください。
- 設問・解説の内容は典型例を示すものであり、すべての症例・状況に当てはまるとは限りません。実臨床では本人・家族の意向と多職種カンファレンスを踏まえた個別判断が必要です。
